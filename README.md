# SameSite Cookie Lab
This project demonstrates SameSite cookie concepts with three sites:
- Main Site
- Sub Main Site (Same-Site scenario)
- Cross Site (Cross-Site scenario)

## How to Use
1. Register and login on the Main Site.
2. Interact with cookies via the provided options.
3. Explore SameSite behavior in Sub Main Site.
4. Observe cross-site behavior in the Cross Site.

## Setup
1. Run `docker-compose up` to start all services.
2. Access the sites on `localhost:8080`, `localhost:8081`, and `localhost:8082`.
